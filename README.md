Cek cuaca di kotamu!: Aplikasi Cuaca Modern dengan React & Tailwind
Cek cuaca di kotamu adalah aplikasi web progresif yang menyajikan data cuaca real-time dari seluruh dunia dengan antarmuka pengguna yang memukau. Dibangun menggunakan React, TypeScript, dan Tailwind CSS, aplikasi ini menggabungkan fungsionalitas tingkat tinggi dengan desain Glassmorphism yang estetis.

✨ Fitur Unggulan:
Deteksi Lokasi Otomatis (Geolocation): Menggunakan Web API browser untuk menyajikan cuaca di lokasi pengguna saat ini secara instan.
Background Dinamis & Animasi: Latar belakang aplikasi berubah secara cerdas mengikuti kondisi cuaca (Cerah, Berawan, Hujan, Salju) lengkap dengan animasi partikel CSS yang interaktif.
Waktu Lokal Internasional: Menghitung dan menampilkan waktu lokal yang akurat di kota tujuan menggunakan offset zona waktu dari API.
Data Cuaca Komprehensif: Menyajikan informasi detail mulai dari suhu (Celcius/Fahrenheit), kelembapan, kecepatan angin, suhu "terasa seperti" (feels like), hingga waktu matahari terbenam.
Prakiraan 5 Hari: Grafik harian yang rapi untuk memantau tren cuaca ke depan.
Pengalaman Pengguna (UX) Premium: Dilengkapi dengan Skeleton Loading untuk transisi data yang halus dan Custom Error Handling yang ramah pengguna.

🛠️ Teknologi yang Digunakan:
React (Vite): Untuk struktur komponen yang cepat dan reaktif.
TypeScript: Menjamin keamanan tipe data dan meminimalkan bug saat pengembangan.
Tailwind CSS: Memberikan fleksibilitas desain modern dan responsif di berbagai perangkat.
OpenWeatherMap API: Sumber data cuaca global yang tepercaya.
Lucide Icons / Heroicons: Untuk representasi visual yang bersih.

🚀 Cara Menjalankan Project:
Clone repository: git clone https://github.com/username/weather-app.git

Install dependencies: npm install

Tambahkan API Key di file .env.local: VITE_WEATHER_API_KEY=your_api_key_here

Jalankan aplikasi: npm run dev

Project ini mendemonstrasikan kemampuan saya dalam mengelola API eksternal, manipulasi state yang kompleks, serta implementasi desain modern yang mengutamakan performa dan aksesibilitas.
